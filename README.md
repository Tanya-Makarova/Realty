# Realty  
Мной разработано консольное приложение для автоматизации работы риэлторского агентства. Программа позволяет управлять объектами недвижимости (квартиры и дома), обеспечивая удобное добавление, редактирование, поиск и анализ данных.   
Функции:  
•	добавление объектов недвижимости (квартиры и дома);  
•	просмотр полной информации обо всех объектах;  
•	расчёт цены за квадратный метр;  
•	формирование текстовых объявлений для объектов;  
•	изменение стоимости объекта;  
•	поиск объектов по адресу;  
•	фильтрацию по диапазону цен;  
•	расчёт прибыли с учётом комиссии и налога;  
•	вывод статистики (общее количество объектов, средняя цена, средняя площадь, максимальная и минимальная цена за м²);  
•	удаление объектов.  
Программа имеет текстовое меню (с автоматическим очищением для удобства) с разделением на логические блоки: просмотр, управление, финансовые операции, поиск и статистика.
